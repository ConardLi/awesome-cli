# awesome-cli

awesome cli tools （有趣又实用的命令行工具）

## cli 工具

```js
npm i conard --g
```

- `cc` 代码复杂度统计

- `cl` 代码行数统计

切换到项目目录下，执行：

```js
conard cc
```

![](./dist/img/cc.png)


```js
conard cl
```

![](./dist/img/cl.jpg)
